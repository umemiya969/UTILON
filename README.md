## üß† Konsep Utama

**Proof of Useful Computation (PoUC)** adalah ide bahwa:
- Komputasi yang mahal (*costly*) tetap diperlukan sebagai bukti.
- Namun hasil komputasi tersebut memiliki nilai praktis (misalnya: optimisasi, pemecahan masalah, analisis data, dsb).
- Verifikasi hasil harus jauh lebih murah dibandingkan proses komputasinya.

UTILON mencoba memformalkan dan mengimplementasikan konsep ini dalam bentuk kode nyata.
